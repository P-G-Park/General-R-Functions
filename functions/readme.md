###lab_value_tidy.R

function for SNUH lab datas (esp. big datas. e.g. SUPREME CDW)

**'0.06 재검중'** -> 0.06

**'재검중 0.902'** -> 0.902

**'재검중 0.02 재검'** -> 0.2

**'.'** -> *NA*

**'QNS'** -> *NA*

**'64 재검'** -> 64
